
#bus tracking

1) Create a virtual environment in the root directory.
Command in root directory =  python -m venv venv

2) Activate your virtual environment using
Command in root directory in Windows =  venv\Scripts\activate 

3) Switch to the bus_tracking directory
Command in root directory = cd bus_tracking

4) Install the packages required for the project using the command.
Command in bus_tracking directory = pip install -r requirements.txt
